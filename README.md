# c-logic-tracing-functions
Conjunto de 10 funções em C projetadas para testar lógica de programação e rastreamento de código. Explora manipulação de dígitos (operador módulo), condicionais aninhadas, operadores ternários e estruturas de decisão complexas.
### 🧠 O que este código exercita?
- **Aritmética Inteira:** Manipulação de números através de restos (`%`) e divisões (`/`) para isolar dígitos específicos.
- **Estruturas Condicionais:** - `switch-case` com comportamento de *fall-through*.
  - `if-else` aninhados e encadeados.
  - Operador Ternário (`? :`).
- **Lógica Booleana:** Uso de operadores `&&` (AND) e `||` (OR) para validações múltiplas.
- **Precedência de Operadores:** Ordem de execução de cálculos dentro de atribuições.

### 🛠️ Como usar
O código já inclui uma função `main` que executa todas as funções com uma entrada de teste (matrícula). Basta compilar com qualquer compilador C (GCC, Clang, etc):

```bash
gcc main.c -o desafios
./desafios
